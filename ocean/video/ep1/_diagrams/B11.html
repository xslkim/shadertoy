map：把多层波浪累加成高度场

1
大涌浪
低频率 · 高振幅 —— 定轮廓
2
中等的波
频率 ×1.9 · 振幅 ×0.22
3
细碎涟漪
频率更高 · 振幅更小 —— 加细节

完整海面
float map(vec3 p) { vec2 uv = p.xz; float freq = SEA_FREQ, amp = SEA_HEIGHT; float h = 0.0; for (int i = 0; i < ITER; i++) { h += sea_octave(uv*freq, choppy) * amp; uv *= octave_m; // 旋转，错开各层 freq *= 1.9; // 频率变高 amp *= 0.22; // 振幅变小 } return p.y - h; // 点到水面的高度差 }
返回值的含义
> 0：点在水面之上　·　< 0：点在水面之下　·　= 0：正好在水面
