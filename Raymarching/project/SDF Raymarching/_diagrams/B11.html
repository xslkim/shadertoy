法线：从 SDF 提取表面朝向
法线 = SDF 的梯度方向
SDF 增长最快的方向 = 离开表面最快的方向 = 法线
vec3 calcNormal(in vec3 pos) { vec3 n = vec3(0.0); for (int i = ZERO; i < 4; i++) { // 位运算生成四面体 4 个顶点方向 vec3 e = 0.5773 * (2.0*vec3( ((i+3)>>1)&1, (i>>1)&1, i&1)-1.0); n += e * map(pos + 0.0005*e).x; } return normalize(n); }
为什么用循环而不展开？
防止编译器把体积庞大的 map() 内联 4 次，严重拖累性能
地板优化
材质 ID < 1.5 时直接返回 (0,1,0)，省去 4 次 map() 调用
