C++ 宿主程序：只需做 4 件事
1
初始化 GLFW / OpenGL
创建窗口与渲染上下文
glfwInit(); window = glfwCreateWindow(...); glfwMakeContextCurrent(window);
2
全屏三角形 trick
不需要任何顶点缓冲（VBO）
// vertex shader 用 gl_VertexID 生成坐标 glDrawArrays(GL_TRIANGLES, 0, 3);
3
每帧上传 Uniform
分辨率 / 时间 / 鼠标
glUniform2f(iRes, width, height); glUniform1f(iTime, glfwGetTime()); glUniform4f(iMouse, mx, my, bx, by);
4
热重载 Shader
改完立刻看到效果
// 每帧检测 shader.glsl 修改时间 // 有变化 → 重新编译并链接，无需重启
