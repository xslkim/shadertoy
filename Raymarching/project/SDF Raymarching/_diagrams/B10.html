包围盒加速（AABB）
点到包围盒的距离 > 当前最近距离 → 整组物体直接跳过
25+ 个几何体，一次 map() 只需计算少数几组
vec2 map(in vec3 pos) { // 地板：pos.y 即到 y=0 平面的距离 vec2 res = vec2(pos.y, 0.0); // 第一组：先做包围盒检测 if (sdBox(pos - g1Center, g1Extents) < res.x) { res = opU(res, vec2( sdSphere(pos - vec3(-2,.25,0), .25), 26.9 // 材质 ID )); // ... 组内更多物体 } // 第二组、第三组 ... 结构相同 return res; // vec2(距离, 材质ID) }
