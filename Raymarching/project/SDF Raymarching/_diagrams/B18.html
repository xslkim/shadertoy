抗锯齿、雾效与 Gamma 校正
超采样抗锯齿
每像素在子像素位置采样 2×2 次
AA 宏控制质量，低端硬件可关闭
for (int m=0; m<AA; m++) for (int n=0; n<AA; n++) col += render(ro, rdOffset(m,n)); col /= float(AA*AA); // 取平均
大气透视雾效
距离 t 的三次方指数衰减
远处物体自然消隐，无生硬截断
// 距离越远，混入背景色越多 col = mix(col, bgColor, 1.0 - exp(-0.0001*t*t*t));
Gamma 校正
线性空间光照 → sRGB 显示空间
不做校正画面会偏暗、颜色失真
// 显示器假设输入是 gamma 2.2 编码 col = pow(col, vec3(1.0 / 2.2));
