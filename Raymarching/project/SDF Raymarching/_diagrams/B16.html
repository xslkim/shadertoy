相机系统：正交基与射线生成
三轴两两正交
cross 叉积保证 uu ⊥ ww、vv ⊥ ww 且 vv ⊥ uu，构成完整相机坐标系
// 相机绕 y 轴旋转 float an = 0.5 * iTime; vec3 ro = vec3(3.5*sin(an), 1.0, 3.5*cos(an)); // 构建正交基（lookAt） vec3 ww = normalize(target - ro); // forward vec3 uu = normalize(cross(ww, UP)); // right vec3 vv = cross(uu, ww); // up（自动正交） // 生成每个像素的射线方向 // p = 像素偏移 / 屏幕高度（归一化） vec3 rd = normalize(p.x*uu + p.y*vv + ww*1.8);
